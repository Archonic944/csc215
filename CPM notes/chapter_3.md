# Chapter 3: The CP/M Based Computer

## Logical devices

- CON = console
- LPT = line printer
- LST = list device
- A,B,C,D = drives
- RDR = reader
- PUN = punch (or general output)

## IO devices

- IOBYT device selectors (bit patterns) determine which device to use
- Can be manually selected or set in the program
